# 代码仓库结构

文档目录说明：

```
├── README.md                   - README
├── docs                        - 服务文档相关文件
│   └── index.md
├── resources                   - 服务资源文件
│   ├── icons
│   │   └── css                 - 样式文件目录
│   │       └── custom.css      - 自定义样式
│   │   └── kodbox_log.png      - 服务logo 
├── ros_templates               - 服务ROS模板，可以有多个
│   └── template.yaml           - 示例ROS模板
├── config.yaml                 - 服务配置文件
```
